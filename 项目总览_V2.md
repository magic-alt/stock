# A股监控与回测系统 V2.0 - 项目总览

## 🎯 项目定位

这是一个**模块化、可扩展**的A股市场监控与策略回测平台，采用现代软件工程架构设计，支持：

- ✅ 多主题股票分组管理（AI、芯片、黄金、新能源等）
- ✅ 实时行情监控与技术指标分析
- ✅ 多策略回测与性能评估
- ✅ 插件式架构，易于扩展
- ✅ 预留专业回测库（backtrader）集成接口

### 设计模式

1. **工厂模式** (Factory Pattern)
   - `DataSourceFactory`: 创建不同类型的数据源
   - 易于扩展新的数据源（tushare、eastmoney等）

2. **策略模式** (Strategy Pattern)
   - `BaseStrategy`: 策略基类
   - 各种具体策略：MA、RSI、MACD等
   - 策略可独立开发、测试、替换

3. **适配器模式** (Adapter Pattern)
   - `BacktraderAdapter`: 适配backtrader库
   - 屏蔽第三方库的复杂性

4. **单一职责原则** (SRP)
   - 每个模块职责明确
   - 数据获取、指标计算、策略逻辑分离

## 📦 模块详解

### 1. 数据源模块 (`src/data_sources/`)

**职责**：提供统一的数据获取接口

```
data_sources/
├── base.py              # 数据源抽象基类
├── akshare_source.py    # AKShare实现
├── factory.py           # 数据源工厂
└── __init__.py
```

**核心类**：
- `DataSource`: 抽象基类，定义标准接口
- `AKShareDataSource`: AKShare实现
- `DataSourceFactory`: 工厂类，创建数据源实例

**扩展方式**：
```python
# 添加新数据源
class TushareDataSource(DataSource):
    def get_stock_realtime(self, stock_code):
        # 实现tushare获取逻辑
        pass
```

### 2. 技术指标模块 (`src/indicators/`)

**职责**：计算各类技术指标

```
indicators/
├── technical.py    # 技术指标计算器
└── __init__.py
```

**支持的指标**：
- MA (移动平均线)
- EMA (指数移动平均)
- RSI (相对强弱指标)
- MACD (平滑异同移动平均线)
- Bollinger Bands (布林带)
- KDJ (随机指标)
- ATR (真实波幅)
- Volume MA (成交量均线)

**使用示例**：
```python
from indicators import TechnicalIndicators

df = TechnicalIndicators.calculate_all_indicators(df)
latest = TechnicalIndicators.get_latest_indicators(df)
```

### 3. 策略模块 (`src/strategies/`)

**职责**：实现各种交易策略

```
strategies/
├── base.py              # 策略基类
├── ma_strategies.py     # 均线策略
├── rsi_strategies.py    # RSI策略
├── macd_strategies.py   # MACD策略
└── __init__.py
```

**策略类型**：
- `MACrossStrategy`: 双均线交叉
- `TripleMACrossStrategy`: 三均线
- `RSIStrategy`: RSI超买超卖
- `RSIDivergenceStrategy`: RSI背离
- `MACDStrategy`: MACD信号
- `MACDZeroCrossStrategy`: MACD零轴

**自定义策略**：
```python
class MyStrategy(BaseStrategy):
    def generate_signals(self, df):
        # 你的策略逻辑
        df['Signal'] = ...
        return df
```

### 4. 回测模块 (`src/backtest/`)

**职责**：执行策略回测

```
backtest/
├── simple_engine.py       # 简单回测引擎
├── backtrader_adapter.py  # Backtrader适配器
└── __init__.py
```

**回测引擎特性**：
- 考虑手续费（万三）
- 考虑印花税（千一）
- 考虑滑点
- 计算夏普比率
- 计算最大回撤
- 详细交易记录

**使用示例**：
```python
from backtest import SimpleBacktestEngine
from strategies import MACrossStrategy

engine = SimpleBacktestEngine(initial_capital=100000)
strategy = MACrossStrategy(5, 20)
results = engine.run(df, strategy)
```

## 🔄 数据流转

### 实时监控流程

```
1. 用户选择监控方案
   ↓
2. StockMonitor初始化
   - 加载watchlist
   - 创建DataSource
   ↓
3. 定时循环
   ↓
4. 获取实时数据
   - 指数数据
   - 股票数据
   ↓
5. 计算技术指标（可选）
   - 获取历史数据
   - 计算指标
   ↓
6. 格式化显示
   - 清屏
   - 打印数据
   ↓
7. 等待下次刷新
```

### 回测流程

```
1. 用户选择股票和策略
   ↓
2. 获取历史数据
   - DataSource.get_stock_history()
   ↓
3. 计算技术指标
   - TechnicalIndicators.calculate_all()
   ↓
4. 生成交易信号
   - Strategy.generate_signals()
   ↓
5. 执行模拟交易
   - BacktestEngine.run()
   - 计算成本
   - 记录交易
   ↓
6. 统计分析
   - 收益率
   - 回撤
   - 胜率
   - 夏普比率
   ↓
7. 展示结果
```

### 添加新策略

```python
# src/strategies/my_strategy.py
from .base import BaseStrategy

class MyCustomStrategy(BaseStrategy):
    def __init__(self, param1, param2):
        super().__init__(name="我的策略")
        self.param1 = param1
        self.param2 = param2
    
    def generate_signals(self, df):
        # 实现策略逻辑
        # 计算信号
        df['Signal'] = 0
        
        # 买入条件
        df.loc[condition_buy, 'Signal'] = 1
        
        # 卖出条件
        df.loc[condition_sell, 'Signal'] = -1
        
        # 生成交易信号
        df['Position'] = df['Signal'].diff()
        
        return df

# 在 __init__.py 中导出
from .my_strategy import MyCustomStrategy
```

# Sample Data

This directory contains small deterministic fixtures for first-run demos and documentation. The data is synthetic, A-share shaped OHLCV data, not investment data and not a recommendation.

## Files

- `a_share_demo_ohlcv.csv`: two-symbol daily OHLCV fixture used by `examples/one_click_demo.py`.

The fixture keeps first-run examples independent from AKShare, TuShare, broker SDKs, and network access.

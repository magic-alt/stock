# 📊 数据库预览快速参考

## 🚀 快速开始

### 命令行快速预览
```bash
# 查看数据汇总
python data_manager.py preview

# 查看指定股票
python data_manager.py preview --type stock --symbol 000001

# 查看指定指数  
python data_manager.py preview --type index --symbol 000001

# 查看更新记录
python data_manager.py preview --type updates
```

### 主程序预览
```bash
python main.py
# 选择: 3. 数据管理 → 6. 数据库预览
```

## 🛠️ 常用操作

### 检查数据是否存在
```powershell
python -c "import sqlite3; conn = sqlite3.connect('datacache/stock_data.db'); cursor = conn.cursor(); cursor.execute('SELECT COUNT(*) FROM stock_history'); print(f'股票数据: {cursor.fetchone()[0]} 条'); conn.close()"
```

### 下载新数据
```bash
# 下载预定义股票（34只）
python data_manager.py download --preset basic

# 更新现有数据
python data_manager.py update

# 查看缓存信息
python data_manager.py info
```

### DB Browser for SQLite
1. 下载: https://sqlitebrowser.org/
2. 打开: `datacache/stock_data.db`
3. 选择 "Browse Data" 选项卡
4. 选择表: `stock_history` 或 `index_history`

## 🔍 故障排查

### 问题: 看不到数据
1. 确认数据库路径: `datacache/stock_data.db`
2. 运行检查脚本: `python test_db_preview.py`
3. 查看预览: `python data_manager.py preview`

### 问题: 数据过时
```bash
python data_manager.py update
```

### 问题: 需要更多股票
```bash
# 下载自定义股票（逗号分隔）
python data_manager.py download --symbols "600000,600519,000001"
```

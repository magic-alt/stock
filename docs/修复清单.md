### 修复2：交易股数和成本计算

**文件：** `src/backtest/simple_engine.py`  
**位置：** `execute_trade()` 方法

#### 2.1 股数计算
```python
# 原代码
shares = self.capital / price

# 修复后
max_shares = self.capital / price
shares = int(max_shares / 100) * 100  # 向下取整到100的整数倍

if shares < 100:  # 资金不足1手
    return
```

#### 2.2 成本扣除
```python
# 原代码
self.position = shares
self.capital -= cost  # ❌ 只扣除手续费

# 修复后
actual_amount = shares * price
cost = self.calculate_cost(price, shares, True)
self.capital -= (actual_amount + cost)  # ✅ 扣除本金+手续费
```

#### 2.3 盈亏计算
```python
# 原代码
profit = (price - buy_trade['price']) * shares - cost - buy_trade['cost']

# 修复后
buy_amount = buy_trade['price'] * shares
sell_amount = actual_amount
total_cost = cost + buy_trade['cost']
profit = sell_amount - buy_amount - total_cost
```

**测试：**
```python
价格: 282.00 元
理论最大股数: 354.61
实际交易股数: 300 (3手)
验证: True (100的整数倍)
```

### 2. 新策略：Donchian 通道突破 ✅

**策略逻辑：**
- **入场**：收盘价突破 N 日最高价 → 买入
- **出场**：收盘价跌破 exit_N 日最低价 → 卖出

**适用场景：**
- 趋势市场（单边上涨/下跌）
- 大周期交易（日线以上）
- 避免震荡市使用

**关键特性：**
```python
class DonchianBreakoutStrategy(BaseStrategy):
    def __init__(self, n: int = 20, exit_n: int = 10):
        # 计算 N 日最高/最低
        df['HH'] = df['最高'].rolling(n, min_periods=n//2).max().shift(1)
        df['LL'] = df['最低'].rolling(exit_n, min_periods=exit_n//2).min().shift(1)
        
        # 突破信号
        df.loc[df['收盘'] > df['HH'], 'Signal'] = 1   # 买入
        df.loc[df['收盘'] < df['LL'], 'Signal'] = -1  # 卖出
```

**健壮性设计：**
- ✅ `min_periods` 避免初期全 NaN
- ✅ `shift(1)` 避免未来函数（使用前一日通道）
- ✅ 参数验证（`validate_parameters()`）
- ✅ 通道宽度计算（`get_channel_width()`）

def normalize_symbol(code: str) -> str:
    """统一代码：6位数字 或 带后缀的代码"""
    code = (code or "").strip().upper()
    if len(code) == 8 and code.endswith((".SH", ".SZ")):
        return code
    return code[:6]

**核心逻辑：**
```python
class SimpleBacktestEngine:
    def __init__(self, ..., 
                 risk_per_trade: float = 0.01,     # 每笔风险1%
                 atr_mult_sl: float = 2.0,         # ATR×2止损
                 atr_trailing: bool = True,        # 移动止损
                 take_profit_pct: float = None):   # 止盈%
        ...
    
    def _position_size_by_risk(self, price, atr):
        """按风险预算计算仓位"""
        per_share_risk = self.atr_mult_sl * atr
        risk_budget = self.capital * self.risk_per_trade
        shares = int((risk_budget / per_share_risk) // 100) * 100
        return max(0, shares)
```

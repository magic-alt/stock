## 🎯 实施内容

### 1. 核心功能开发

#### 1.1 命令行预览工具
**文件**: `data_manager.py`

新增函数:
- `cmd_preview_data()` - 主预览命令处理器
- 支持5种预览类型: summary, stock, index, updates, recent

参数支持:
```bash
--type {summary,stock,index,updates,recent}
--symbol SYMBOL  # 股票/指数代码
--rows ROWS      # 显示行数（默认20）
```

**测试验证**:
```bash
✅ python data_manager.py preview
✅ python data_manager.py preview --type stock --symbol 000001
✅ python data_manager.py preview --type index --symbol 000001
✅ python data_manager.py preview --type updates
```

#### 1.2 主程序交互式预览
**文件**: `main.py`

新增函数:
- `preview_database()` - 主预览入口
- `preview_stock_summary()` - 股票汇总视图
- `preview_index_summary()` - 指数汇总视图
- `preview_recent_updates()` - 更新记录视图
- `preview_specific_stock()` - 单个股票详情
- `preview_specific_index()` - 单个指数详情

## 🎯 问题解决

### 原始问题
用户报告: "在DB Browser for SQLite中查看stock_data.db没有数据"

### 问题诊断
经过排查发现：
1. ✅ 数据库文件存在: `datacache/stock_data.db`
2. ✅ 数据确实存在: 1,021条股票数据，210条指数数据
3. ⚠️ 可能原因: 用户打开了错误的数据库文件（测试数据库）

### 解决方案
1. **明确数据库位置**: 在所有文档中强调正确路径
2. **提供预览工具**: 无需第三方工具即可查看数据
3. **快速验证命令**: 提供一行命令检查数据
4. **详细文档**: 完整的使用指南和故障排查

## 🎯 5分钟快速上手

### 步骤1: 配置数据（📊 标签页）
```
1. 数据源: akshare (默认)
2. 股票代码: 
   600519.SH
   000333.SZ
   600036.SH
3. 开始日期: 2022-01-01
4. 结束日期: 2024-12-31
5. 基准指数: 000300.SS
6. 点击"预览数据"验证
```

### 步骤2: 选择策略（🎯 标签页）
```
1. 点击"趋势策略"按钮
   (自动选择 ema, macd, adx_trend 等)
2. 策略参数留空（使用默认）
```

### 步骤3: 配置回测（⚙️ 标签页）
```
1. 初始资金: 200000
2. 手续费率: 0.0001
3. 其他保持默认
4. ☑ 生成图表
```

### 步骤4: 设置优化（🔍 标签页）
```
1. 运行模式: ○ 自动化流程
2. 并行进程数: 4
3. Top N: 5
4. ☑ Hot-Only 模式
```

### 步骤5: 开始回测
```
点击 ▶️ 开始回测 按钮
等待 1-3 分钟
查看右侧日志输出
```

### 步骤6: 查看结果
```
reports_gui/
  ├── pareto_front.png      (Pareto前沿图)
  ├── opt_*.csv             (优化结果)
  └── rerun_rank_1/         (最优配置详情)
      ├── summary.json
      ├── nav_series.csv
      └── backtest_plot.png
```

## 💡 使用技巧

### 技巧1: 快速股票选择
点击预设按钮，无需手动输入：
- **白酒股**: 茅台、五粮液等
- **银行股**: 工行、建行等  
- **科技股**: 用友、恒生等

### 技巧2: 策略组合
推荐组合：
- **保守型**: ema + macd + triple_ma
- **激进型**: rsi + bollinger + zscore
- **平衡型**: ema + macd + bollinger + rsi

### 技巧3: 参数优化
网格搜索示例：
```json
{
  "period": [10, 20, 30],
  "threshold": [0.01, 0.02]
}
```
会测试 3×2=6 种组合

### 技巧4: 性能优化
- 股票数 ≤ 10 只
- 策略数 ≤ 8 个
- 并行进程 = CPU核心数 - 1
- 启用 Hot-Only 模式

### 技巧5: 配置复用
常用配置保存为文件：
- `config_liquor.json` - 白酒股配置
- `config_bank.json` - 银行股配置
- `config_tech.json` - 科技股配置

## ❓ 常见问题 FAQ

### Q: GUI 和 CLI 有什么区别？
**A**: 功能完全相同，GUI 更易用，CLI 更适合脚本化。

### Q: 可以同时运行多个实例吗？
**A**: 可以，但注意输出目录要设置不同。

### Q: 配置文件可以手动编辑吗？
**A**: 可以，是标准 JSON 格式。

### Q: Hot-Only 模式是什么？
**A**: 只搜索经过验证的参数范围，更快更稳定。

### Q: 为什么有些策略没有结果？
**A**: 可能数据不足或参数不适合，查看日志了解详情。

### Q: 可以添加自定义策略吗？
**A**: 可以，在 `src/strategies/` 添加后需要在代码中注册。

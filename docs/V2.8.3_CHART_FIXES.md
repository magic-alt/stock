虽然 GUI 有"生成图表"复选框，但实际运行时无法正确生成和保存图表。

### 3. 买卖点标记不可见
Figure 0 中应该显示的 ▲ (买入) 和 ▼ (卖出) 标记不可见或太小。

---

## 根本原因分析

### 问题 1: 空白 Figure 1
**原因**: Backtrader 的 `plot()` 方法在某些情况下会创建多个 figure，但只有第一个包含数据。当使用 `plt.savefig()` 保存时，可能保存了错误的 figure。

**代码问题**:
```python
# 旧代码 (src/backtest/plotting.py)
figs = cerebro.plot(**plot_kwargs)
plt.savefig(out_file, dpi=300, bbox_inches='tight')  # 保存当前活动 figure（可能是空白的）
```

### 问题 2: Unicode 编码错误
**原因**: Windows PowerShell 默认使用 GBK 编码，无法显示 Unicode 符号（✓、✗、❌等）。

**错误信息**:
```
UnicodeEncodeError: 'gbk' codec can't encode character '\u2713' in position 0
```

### 问题 3: 买卖点标记不可见
**原因**: Backtrader 的默认标记样式可能太小或配置不正确，需要手动添加明显的标记。

---

## 解决方案

### 修复 1: 只保存有数据的图表

**文件**: `src/backtest/plotting.py`  
**行数**: ~390-400

**修改前**:
```python
if out_file:
    plt.savefig(out_file, dpi=300, bbox_inches='tight')
    print(f"✓ 图表已保存到: {out_file}")
    plt.close()
```

**修改后**:
```python
if out_file:
    # 关闭所有空白图表，只保存第一个
    if figs and len(figs) > 0:
        fig_to_save = figs[0][0]
        fig_to_save.savefig(out_file, dpi=300, bbox_inches='tight')
        print(f"[OK] 图表已保存到: {out_file}")
        # 关闭所有图表
        plt.close('all')
    else:
        plt.savefig(out_file, dpi=300, bbox_inches='tight')
        print(f"[OK] 图表已保存到: {out_file}")
        plt.close('all')
```

**效果**: 确保只保存包含数据的主图表，避免空白 figure。

### 修复 3: 手动添加买卖点标记

**文件**: `src/backtest/plotting.py`  
**行数**: ~320-380

**新增代码**:
```python
# 手动添加买卖点标记（因为 Backtrader 默认标记可能不明显）
try:
    if figs and len(figs) > 0:
        fig = figs[0][0]  # 获取第一个图表
        axes = fig.get_axes()
        
        # 获取策略实例以访问交易记录
        if cerebro.runstrats and len(cerebro.runstrats) > 0:
            strat = cerebro.runstrats[0][0]
            
            # 收集买卖点数据
            buy_dates = []
            buy_prices = []
            sell_dates = []
            sell_prices = []
            
            # 从订单记录中提取买卖点
            if hasattr(strat, '_orders'):
                for order in strat._orders:
                    if order.status == order.Completed:
                        exec_date = bt.num2date(order.executed.dt)
                        price = order.executed.price
                        
                        if order.isbuy():
                            buy_dates.append(exec_date)
                            buy_prices.append(price)
                        else:
                            sell_dates.append(exec_date)
                            sell_prices.append(price)
            
            # 在第一个子图（价格图）上绘制标记
            if len(axes) > 0 and (buy_dates or sell_dates):
                price_ax = axes[0]  # 第一个子图是价格图
                
                # 绘制买入点（红色向上三角形）
                if buy_dates:
                    price_ax.scatter(
                        buy_dates, buy_prices,
                        marker='^',
                        color='red',
                        s=200,  # 大小
                        alpha=0.9,
                        edgecolors='darkred',
                        linewidths=2,
                        zorder=5,  # 确保在其他元素上方
                        label='买入'
                    )
                
                # 绘制卖出点（绿色向下三角形）
                if sell_dates:
                    price_ax.scatter(
                        sell_dates, sell_prices,
                        marker='v',
                        color='lime',
                        s=200,  # 大小
                        alpha=0.9,
                        edgecolors='darkgreen',
                        linewidths=2,
                        zorder=5,  # 确保在其他元素上方
                        label='卖出'
                    )
                
                # 添加图例
                if buy_dates or sell_dates:
                    price_ax.legend(loc='upper left', fontsize=9, framealpha=0.8)
                    print(f"[OK] 已添加买卖点标记: {len(buy_dates)} 个买入, {len(sell_dates)} 个卖出")

except Exception as e:
    print(f"[警告] 添加买卖点标记失败: {e}")
    pass  # 标记失败也继续
```

**效果**: 
- 买入点: 红色向上三角形 (▲), 大小 200, 深红色边框
- 卖出点: 亮绿色向下三角形 (▼), 大小 200, 深绿色边框
- 添加图例说明
- 确保标记在所有元素上方 (zorder=5)

## 常见问题

### Q1: 图表仍然显示空白？
**A**: 检查以下事项：
1. 确认使用了 `--plot` 参数
2. 验证数据已下载：`ls cache/`
3. 检查输出目录权限：`mkdir -Force test_auto_reports`

### Q2: 买卖点标记重叠？
**A**: 可能因为交易过于密集，建议：
- 减少交易频率（调整策略参数）
- 增大图表尺寸：`figsize=(20, 12)`
- 使用交互式查看：移除 `--out_dir` 参数

### Q3: GUI 中如何查看图表？
**A**: 
1. 勾选"生成图表"复选框
2. 运行回测
3. 打开输出目录（默认 `./reports/`）
4. 双击 `*_chart.png` 文件

### Q4: 如何只显示买入点不显示卖出点？
**A**: 修改 `plotting.py` 中的 scatter 代码，注释掉卖出部分。

### Matplotlib 配置

```python
# 使用 Agg 后端（非 GUI）
matplotlib.use('Agg')

rcParams['font.sans-serif'] = ['SimHei', 'DejaVu Sans']

```

### Backtrader Plot 方案
```python
class CNPlotScheme(PlotScheme):
    def __init__(self):
        super().__init__()
        self.barup = 'red'          # 阳线
        self.bardown = 'green'      # 阴线
        self.volup = 'red'          # 成交量
        self.voldown = 'green'
        self.grid = True
        self.gridstyle = '--'
        self.gridalpha = 0.25
```

### 手动标记实现
使用 `matplotlib.axes.Axes.scatter()` 方法直接在价格子图上添加标记点，而不依赖 Backtrader 的内置标记系统。

**优势**:
- 完全控制样式
- 保证可见性
- 支持自定义图例
- 不受 Backtrader 配置限制

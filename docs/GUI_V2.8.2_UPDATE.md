## 📋 更新内容

根据用户反馈，本次更新新增三项功能优化：

### 1. ✅ 单只股票快速选择

**需求**: "gui中股票没法选择单只请优化"

**实现**:

在数据配置页面的"快速选择"区域新增单只股票按钮：

```
快速选择
┌─────────────────────────────────────────────┐
│ 单股: [茅台] [平安] [招行] [五粮液]          │
│ 组合: [白酒股] [银行股] [科技股] [清空]      │
└─────────────────────────────────────────────┘
```

**股票映射**:
- **茅台**: 600519.SH (贵州茅台)
- **平安**: 601318.SH (中国平安)
- **招行**: 600036.SH (招商银行)
- **五粮液**: 000858.SZ (五粮液)

**使用方法**:
1. 点击任意单股按钮（如"茅台"）
2. 股票代码输入框自动填充该股票代码
3. 可直接进行单只股票回测分析

**代码位置**: `backtest_gui.py` 第176-239行

---

### 2. ✅ 下载数据功能

**需求**: "没有下载数据选项 请优化"

**实现**:

在数据配置页面新增"下载数据"按钮，与"预览数据"并列：

```
[📥 下载数据]  [📋 预览数据]
```

**功能特性**:
- 批量下载股票数据到缓存
- 逐个下载并显示进度
- 统计成功/失败数量
- 自动下载基准指数数据
- 显示每只股票的数据记录数和时间范围

**输出示例**:
```
============================================================
开始下载数据...
📥 数据源: akshare
📊 股票代码: 600519.SH, 000858.SZ
📅 日期范围: 2022-01-01 ~ 2024-12-31
💾 缓存目录: ./cache

[1/2] 下载 600519.SH...
  ✓ 成功: 687 条记录 (2022-01-04 ~ 2024-12-31)
[2/2] 下载 000858.SZ...
  ✓ 成功: 687 条记录 (2022-01-04 ~ 2024-12-31)

下载基准指数: 000300.SH...
  ✓ 基准指数下载成功: 687 条记录

============================================================
✓ 下载完成! 成功: 2, 失败: 0
```

**使用场景**:
- 首次使用前批量下载数据
- 更新历史数据
- 预先缓存数据避免回测时等待
- 检查数据可用性

**代码位置**: `backtest_gui.py` 第790-867行

---

### 3. ✅ 图表生成选项

**需求**: "CLI回测中--plot选项也要加入到gui中的配置，最终的图表也要保存下来"

**状态**: ✅ 已存在且正常工作

**确认**:
- 图表选项已在"回测配置"标签页中存在
- 选项名称: "生成图表"
- 默认状态: ✅ 开启
- 实现位置: `backtest_gui.py` 第449-454行

**回测引擎调用**:
```python
# 单次回测
result = engine.run_strategy(
    ...
    enable_plot=self.plot_var.get()  # ✅ 已传递
)

# 网格搜索
results = engine.run_grid_search(
    ...
    enable_plot=self.plot_var.get()  # ✅ 已传递
)
```

**图表保存位置**:
- 输出目录: `reports_gui/` (可配置)
- 文件命名: `{strategy}_chart.png`
- 示例: 
  - `ema_chart.png`
  - `macd_chart.png`
  - `bollinger_chart.png`

**图表内容**:
- 股票价格走势
- 买卖信号标记
- 净值曲线
- 回撤曲线
- 交易统计信息

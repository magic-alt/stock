# 佣金计算详解

## 问题总结

您提出了两个关键问题：

### 1. Size单位是什么？

**答案**: **Size = 股数，但系统强制100股整数倍**

- Backtrader中，`size=100` 表示 **100股 = 1手**
- 中国A股规则：最小交易单位为1手（100股）
- **系统已强制**: 所有交易必须是100股的整数倍

**示例**:
```python
self.buy(size=100)   # ✅ 买入100股 = 1手（允许）
self.buy(size=200)   # ✅ 买入200股 = 2手（允许）
self.buy(size=1)     # ❌ 买入1股（系统会自动调整或拒绝）
self.buy(size=150)   # ❌ 买入150股（系统会调整为100股）
```

**重要**: 系统已在`CNStockCommission`类中实现了100股整数倍的强制规则。

### 2. 佣金"免五"的正确理解

**错误理解**（之前的实现）:
- ❌ 免五 = 佣金最低5元

**正确理解**:
- ✅ **免五** = 按实际佣金率收取，**无最低限制**
- ❌ **不免五** = 佣金不足5元时按5元收取

## 修正后的实现

### 代码修改（src/backtest/engine.py）

```python
class CNStockCommission(bt.CommInfoBase):
    """中国A股佣金和印花税计算"""
    params = (
        ('commission', 0.0001),      # 佣金率：万分之一
        ('stamp_duty', 0.0005),      # 印花税：万分之五（仅卖出）
        ('min_commission', 0.0),     # 最低佣金：0元（免五模式）
    )
    
    def _getcommission(self, size, price, pseudoexec):
        """计算佣金
        
        免五模式（min_commission=0）：按实际佣金率收取
        不免五模式（min_commission=5）：佣金不足5元按5元收取
        """
        value = abs(size) * price
        comm = value * self.p.commission
        
        # 如果设置了最低佣金（不免五）
        if self.p.min_commission > 0 and comm < self.p.min_commission:
            comm = self.p.min_commission
        
        return comm
    
    def getcommission(self, size, price):
        """获取总交易成本（佣金 + 印花税）"""
        # 佣金（买卖都收）
        comm = self._getcommission(size, price, False)
        
        # 印花税（仅卖出收取）
        stamp = 0.0
        if size < 0:  # 卖出时 size 为负
            stamp = abs(size) * price * self.p.stamp_duty
        
        return comm + stamp
```

### 成本分解

#### 买入费用
```
Commission = 佣金
           = 交易金额 × 0.0001
```

#### 卖出费用
```
Commission = 佣金 + 印花税
           = 交易金额 × 0.0001 + 交易金额 × 0.0005
           = 交易金额 × 0.0006
```

## 为什么选择"免五"作为默认？

### 优势
1. ✅ **精确计算**: 按实际交易金额比例收费
2. ✅ **适合回测**: 更接近实际大资金账户成本
3. ✅ **支持小单**: 可以测试1股、10股等小额交易
4. ✅ **现代券商**: 大部分互联网券商都是免五

### 不免五的问题
1. ❌ **小额交易失真**: 1股交易收5元，实际费率高达0.28%
2. ❌ **不适合回测**: 会严重低估策略收益
3. ❌ **计算不准**: 固定费用导致费率随交易金额波动

### 对比示例

| 交易量 | 交易金额 | 免五佣金 | 免五费率 | 不免五佣金 | 不免五费率 |
|--------|----------|----------|----------|------------|------------|
| 1股    | 1,770元  | 0.18元   | 0.010%   | 5.00元     | 0.282% ❌  |
| 10股   | 17,708元 | 1.77元   | 0.010%   | 5.00元     | 0.028%     |
| 100股  | 177,085元| 17.71元  | 0.010%   | 17.71元    | 0.010% ✅  |
| 1000股 | 1,770,850元| 177.09元| 0.010%  | 177.09元   | 0.010% ✅  |

**结论**: 只有交易金额 ≥ 5万元时，不免五才接近万一费率。

## Cost字段说明

您提到"Cost是不是算错了"，让我澄清：

### Cost/Value的含义

- **Cost** (买入时): 交易金额 = Price × Size
- **Value** (卖出时): 交易金额 = Price × Size

**注意**: Cost/Value **不包含**佣金和印花税！

### 完整成本计算（100股示例）

#### 买入100股
```
实际支付 = Cost + Commission
         = 177,085 + 17.71
         = 177,102.71元
```

#### 卖出100股
```
实际收到 = Value - Commission
         = 174,551 - 104.74
         = 174,446.26元
```

#### 盈亏
```
盈亏 = 实际收到 - 实际支付
     = 174,446.26 - 177,102.71
     = -2,656.45元
```

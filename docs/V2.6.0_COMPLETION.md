## 二、技术实现

### 2.1 事件总线 (src/core/events.py)

**核心类**:
```python
@dataclass(slots=True)
class Event:
    type: str
    data: Any = None

class EventEngine:
    def register(self, etype: str, handler: Handler) -> None
    def put(self, event: Event) -> None
    def start(self) -> None  # 后台线程
    def stop(self) -> None
```

**特性**:
- ✅ 线程安全（Queue + Thread）
- ✅ 非阻塞发布（put不等待）
- ✅ 异常隔离（handler错误不影响其他）
- ✅ 优雅关闭（1秒超时）

**标准事件类型**:
```python
class EventType:
    DATA_LOADED = "data.loaded"
    STRATEGY_SIGNAL = "strategy.signal"
    ORDER_FILLED = "order.filled"
    METRICS_CALCULATED = "metrics.calculated"
    PIPELINE_STAGE = "pipeline.stage"
    RISK_WARNING = "risk.warning"
    # ... 20+ 事件类型
```

### 2.3 Engine依赖注入 (src/backtest/engine.py)

**改动1: 构造函数**
```python
class BacktestEngine:
    def __init__(
        self,
        *,
        source: str = "akshare",  # 向后兼容
        benchmark_source: Optional[str] = None,
        cache_dir: str = CACHE_DEFAULT,
        # V2.6.0: 可选依赖注入
        event_engine: Optional[EventEngine] = None,
        history_gateway: Optional[HistoryGateway] = None,
    ) -> None:
        # 默认创建实例（向后兼容）
        self.events = event_engine or EventEngine()
        self.gw = history_gateway or BacktestGateway(source, cache_dir)
```

**改动2: _load_data()**
```python
def _load_data(self, symbols, start, end, adj=None):
    # 使用gateway协议
    data = self.gw.load_bars(symbols, start, end, adj=adj)
    
    # 发布事件
    self.events.put(Event(EventType.DATA_LOADED, {
        "symbols": list(data.keys()),
        "start": start,
        "end": end,
        "count": sum(len(df) for df in data.values())
    }))
    
    return data
```

**改动3: _load_benchmark()**
```python
def _load_benchmark(self, index_code, start, end):
    # 使用gateway的load_index_nav（内置fallback）
    nav = self.gw.load_index_nav(index_code, start, end)
    
    # 发布事件
    self.events.put(Event("benchmark.loaded", {
        "index": index_code,
        "points": len(nav),
        "start_value": float(nav.iloc[0]),
        "end_value": float(nav.iloc[-1])
    }))
    
    return nav
```

**向后兼容性**:
```python
# 旧方式：直接使用（完全兼容）
engine = BacktestEngine(source="akshare")

# 新方式：依赖注入（可选）
events = EventEngine()
gateway = BacktestGateway("akshare")
engine = BacktestEngine(event_engine=events, history_gateway=gateway)
```

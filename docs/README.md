# A股实时监控和回测系统

这是一个用于监控A股市场的Python工具，支持实时行情显示和简单的回测功能。

## 功能特性

### 1. 实时监控
- 显示主要指数（上证、深证、创业板等）的实时行情
- 监控自选股票的实时价格和涨跌幅
- 计算并显示技术指标：
  - 移动平均线（MA5, MA10, MA20）
  - 相对强弱指标（RSI）
  - MACD指标
- 周期性自动刷新数据

### 2. 回测系统
- 基于双均线策略的简单回测
- 计算总收益率和最大回撤
- 显示详细的交易记录
- 支持自定义回测周期和初始资金

## 安装依赖

```bash
pip install -r requirements.txt
```

主要依赖：
- `akshare`: 获取A股市场数据
- `pandas`: 数据处理
- `numpy`: 数值计算

## 使用方法

### 运行程序

```bash
python stock_monitor.py
```

### 配置自选股

编辑 `config.py` 文件中的 `WATCHLIST` 列表，添加你想监控的股票代码：

```python
WATCHLIST = [
    '600519',  # 贵州茅台
    '000858',  # 五粮液
    # 添加更多股票...
]
```

### 功能选择

运行程序后，会出现菜单：

1. **实时监控** - 开始监控指数和自选股票
2. **回测系统** - 进行历史数据回测
0. **退出** - 退出程序

## 技术指标说明

### 移动平均线 (MA)
- MA5: 5日均线，反映短期趋势
- MA10: 10日均线，反映中期趋势
- MA20: 20日均线，反映长期趋势

### RSI (相对强弱指标)
- 范围: 0-100
- > 70: 超买区域
- < 30: 超卖区域

### MACD (指数平滑移动平均线)
- MACD线与Signal线的关系反映买卖时机
- MACD > Signal: 可能的买入信号
- MACD < Signal: 可能的卖出信号

## 回测策略

默认使用**双均线策略**：
- **买入信号**: MA5上穿MA20
- **卖出信号**: MA5下穿MA20

## 注意事项

1. 数据来源于akshare，需要网络连接
2. 实时数据可能有延迟（通常15分钟左右）
3. 回测结果仅供参考，不构成投资建议
4. 交易时间外可能无法获取实时数据
5. 频繁请求可能被限制，建议刷新间隔不低于30秒

## 自定义配置

可以在 `config.py` 中修改：
- 自选股票列表
- 监控的指数
- 刷新间隔
- 技术指标参数
- 回测默认参数

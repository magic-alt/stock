# 回测分析GUI使用指南 V2.10.3

## 概述

量化回测分析系统 GUI V2.10.3 是一个完整的图形界面，支持所有 `unified_backtest_framework.py` 的 CLI 命令。

## 启动方式

```bash
python scripts/backtest_gui.py
```

或使用启动脚本：
```bash
# Windows
.\启动GUI.bat

# Linux/Mac
python scripts/backtest_gui.py
```

## 功能模块

### 1. 🎯 单策略回测 (run命令)

执行单个策略的完整回测流程。

**主要功能**:
- 策略选择（从策略注册表中选择）
- 多股票代码输入（空格分隔）
- 日期范围设置
- 数据源配置（akshare/yfinance/tushare）
- 复权方式选择（前复权/后复权/不复权）
- 基准指数对比
- 策略参数自定义（JSON格式）
- 自动生成图表和Markdown报告

**使用步骤**:
1. 选择标签页：**🎯 单策略回测 (run)**
2. 选择策略（如 `macd`）
3. 输入股票代码（如 `600519.SH 000858.SZ`）
4. 设置日期范围
5. 可选：自定义策略参数（JSON格式）
6. 勾选"生成回测图表和报告"
7. 点击"▶ 运行"

**示例配置**:
```
策略: macd
股票: 600519.SH 000858.SZ
日期: 2023-01-01 到 2024-12-31
策略参数:
{
  "fast": 12,
  "slow": 26,
  "signal": 9
}
```

---

### 2. 🔍 网格搜索 (grid命令)

对单个策略进行参数网格搜索优化。

**主要功能**:
- 参数空间定义（JSON格式）
- 并行计算加速
- 结果导出CSV
- 帕累托最优参数识别

**使用步骤**:
1. 选择标签页：**🔍 网格搜索 (grid)**
2. 选择策略
3. 输入股票代码（通常单只）
4. 定义参数网格（JSON格式）
5. 设置并行Workers数量
6. 指定输出CSV文件
7. 点击"▶ 运行"

**参数网格示例**:
```json
{
  "fast": [10, 12, 15],
  "slow": [26, 30, 35],
  "signal": [9]
}
```

**结果文件**:
- `grid_results.csv`: 包含所有参数组合的性能指标

---

### 3. 🚀 自动优化 (auto命令)

多策略自动优化，找到最佳策略和参数组合。

**主要功能**:
- 预设配置快速启动
- 多策略并行测试
- 帕累托前沿分析
- Top-N最优结果
- 市场态势过滤
- 批量报告生成

**使用步骤**:
1. 选择标签页：**🚀 自动优化 (auto)**
2. 选择预设配置（可选）或手动配置
3. 输入股票代码（多个）
4. 选择策略列表（多选）
5. 设置Top-N数量
6. 勾选高级选项（热门参数、市场态势）
7. 点击"▶ 运行"

**预设配置**:
- **白酒股-趋势策略**: 600519.SH等 + EMA/MACD等趋势策略
- **银行股-震荡策略**: 600036.SH等 + Bollinger/RSI等震荡策略
- **科技股-全策略**: 000333.SZ等 + 多种策略综合

**高级选项**:
- **热门参数区间**: 使用精简的参数网格，加速搜索
- **市场态势过滤**: 使用基准EMA200过滤牛市/熊市

**输出目录**:
```
./reports_auto/
├── summary.csv              # 所有结果汇总
├── pareto_front.csv         # 帕累托最优配置
├── top5_configs.json        # Top 5配置
├── heat_*.png               # 热力图
└── reports/                 # 各策略详细报告
    ├── strategy1_report.md
    ├── strategy1_chart.png
    └── ...
```

### 4. 📥 数据下载

批量下载股票历史数据到本地缓存。

**主要功能**:
- 批量股票代码输入
- 自定义日期范围
- 数据源选择
- 复权方式配置
- 实时下载进度

**使用步骤**:
1. 选择标签页：**📥 数据下载**
2. 输入股票代码（每行一个或空格分隔）
3. 设置日期范围
4. 选择数据源
5. 选择复权方式
6. 点击"🚀 开始下载"

**示例**:
```
股票代码:
600519.SH
000858.SZ
600036.SH
000333.SZ

日期: 2020-01-01 到 2024-12-31
数据源: akshare
复权: 前复权 (qfq)
```

**缓存位置**: `./cache/` 目录

---

### 5. 📋 策略列表 (list命令)

查看所有可用策略及其说明。

**功能**:
- 策略名称和描述
- 默认参数网格
- 参数说明

**使用**:
1. 选择标签页：**📋 策略列表 (list)**
2. 浏览策略信息

**可用策略** (部分):
- `ema`: 指数移动平均线趋势跟踪
- `macd`: MACD指标策略
- `bollinger`: 布林带震荡策略
- `rsi`: RSI超买超卖策略
- `adx_trend`: ADX趋势强度策略
- `triple_ma`: 三重移动平均线策略
- ...（更多请查看策略列表）

## 常见问题

### Q1: 如何查看生成的报告？

**A**: 报告默认保存在以下位置：
- **单策略回测**: `./report_YYYYMMDD_HHMMSS/` 目录
  - `backtest_report.md`: Markdown报告
  - `backtest_chart.png`: 回测图表
  - `backtest_summary.json`: JSON数据
  
- **自动优化**: `./reports_auto/` 目录
  - 查看 `summary.csv` 了解所有结果
  - 查看 `reports/` 子目录中的详细报告

### Q2: 策略参数JSON格式是什么？

**A**: JSON格式示例：
```json
{
  "fast": 12,
  "slow": 26,
  "signal": 9
}
```

每个策略的可用参数不同，可以通过以下方式查看：
1. 点击"策略说明"按钮
2. 切换到"📋 策略列表"标签页
3. 查看策略的默认网格配置

### Q3: 如何加速网格搜索？

**A**: 
1. 增加并行Workers数量（建议设为CPU核心数）
2. 在auto模式下勾选"仅使用热门参数区间"
3. 缩小日期范围（先用短期数据测试）
4. 减少参数网格的组合数量

### Q4: 数据源选择建议？

**A**:
- **A股**: 推荐 `akshare`（免费，无需API Key）
- **美股**: 使用 `yfinance`（免费）
- **港股**: 使用 `akshare`或`yfinance`
- **需要高级数据**: 使用 `tushare`（需要注册获取token）

### Q5: 复权方式如何选择？

**A**:
- **前复权 (qfq)**: 推荐用于回测（保持最新价格不变）
- **后复权 (hfq)**: 适合查看历史真实价格
- **不复权 (noadj)**: 用于特殊分析需求

## 技术说明

### 后台执行机制

GUI使用 `subprocess` 调用 `unified_backtest_framework.py`，所有命令在后台执行：
- 实时输出显示
- 可随时中止任务
- 不阻塞GUI界面

### 配置保存/加载

配置可以保存为JSON文件（功能预留）：
```json
{
  "run": {
    "strategy": "macd",
    "symbols": ["600519.SH"],
    "start": "2023-01-01",
    "end": "2024-12-31",
    ...
  },
  "grid": {...},
  "auto": {...}
}
```

### 错误处理

GUI包含完整的错误处理：
- JSON格式验证
- 必填字段检查
- 数据源可用性检查
- 命令执行错误捕获

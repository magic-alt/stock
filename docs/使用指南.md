# A股监控系统使用指南

## 快速开始

### 方式1: 快速监控（推荐新手）
直接运行简化版实时监控：
```bash
python quick_monitor.py
```
这将立即开始监控默认的几只股票。

### 方式2: 完整功能
运行完整版程序：
```bash
python stock_monitor.py
```
然后选择：
- `1` - 实时监控
- `2` - 回测系统

## 自定义你的股票列表

### 方法1: 修改quick_monitor.py
编辑 `quick_monitor.py` 文件，找到这一行：
```python
WATCHLIST = ['600519', '000858', '601318', '600036']
```
修改为你想监控的股票代码：
```python
WATCHLIST = ['600519', '000001', '300750', '601888']
```

### 方法2: 修改config.py
编辑 `config.py` 文件中的 `WATCHLIST` 列表。

## 使用回测功能

1. 运行 `python stock_monitor.py`
2. 选择 `2` 进入回测系统
3. 输入股票代码（如：600519）
4. 选择是否自定义参数：
   - 不自定义：使用最近一年数据，初始资金10万
   - 自定义：输入开始日期、结束日期、初始资金

### 回测示例
```
请输入股票代码 (如 600519): 600519
使用默认回测周期: 20241009 至 20251009
是否自定义参数? (y/n): n
```

回测结果会显示：
- 总收益率
- 最大回撤
- 交易次数
- 详细交易记录

## 技术指标解读

### 移动平均线（MA）
- **金叉**：短期均线上穿长期均线（看涨信号）
- **死叉**：短期均线下穿长期均线（看跌信号）

### RSI（相对强弱指标）
- **> 70**：超买，可能回调
- **< 30**：超卖，可能反弹
- **50左右**：中性

### MACD
- **MACD > Signal**：上升趋势
- **MACD < Signal**：下降趋势
- **MACD柱状图转正**：可能的买入时机

## 调整刷新频率

编辑脚本中的 `REFRESH_INTERVAL` 变量：
```python
REFRESH_INTERVAL = 30  # 30秒刷新一次
```

建议不低于30秒，避免请求过于频繁。

## 常见问题

### Q: 为什么显示的不是实时价格？
A: A股行情数据通常有15分钟左右的延迟，这是免费数据源的限制。

### Q: 如何添加更多技术指标？
A: 可以修改 `stock_monitor.py` 中的 `calculate_technical_indicators` 函数，添加自己需要的指标计算逻辑。

### Q: 回测策略可以自定义吗？
A: 可以修改 `SimpleBacktest` 类中的 `calculate_signals` 方法，实现自己的交易策略。

### Q: 程序运行出错怎么办？
A: 
1. 先运行 `python test_connection.py` 检查网络和数据源
2. 确认股票代码输入正确
3. 检查是否在交易时间内

### Q: 可以监控港股或美股吗？
A: 当前版本仅支持A股，如需支持其他市场，需要修改数据源接口。

## ✅ 已完成：补丁1 - 交易规则插件化

### 核心改进
- A股交易成本规则（佣金/印花税/整手）已插件化
- Engine 代码减少 82 行，扩展性提升
- 100% 向后兼容，现有代码无需修改

### 默认使用（无需改动）
```python
engine = BacktestEngine()
metrics = engine.run_strategy("ema", ["600519.SH"], "2024-01-01", "2024-12-31")
# 自动应用 A股规则：万一佣金 + 万五印花税 + 100股整手
```

### 自定义配置（可选）
```python
params = {
    "period": 20,
    "_commission": 0.00015,      # 万1.5佣金
    "_min_commission": 5.0,       # 不免五（最低5元）
    "_stamp_tax": 0.0005,         # 万五印花税
    "_lot_size": 100,             # 100股整手
}
metrics = engine.run_strategy("ema", ["600519.SH"], "2024-01-01", "2024-12-31", params=params)
```

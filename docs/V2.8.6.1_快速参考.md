## 🔍 技术细节

### 费率配置 (不变)

| 项目 | 费率 |
|-----|------|
| 买入佣金 | 0.01% |
| 卖出佣金 | 0.01% |
| 卖出印花税 | 0.05% |
| **卖出总费用** | **0.06%** |

### 正确计算公式

```python
# 买入
commission = 交易金额 × 0.0001

# 卖出
total_cost = 交易金额 × 0.0006  # 佣金+印花税
```

## ⚠️ 重要提示

1. **不要使用** `order.executed.comm` 计算总费用
2. **始终从** `order.executed.value` 计算
3. **卖出费用** = 佣金 + 印花税 = 0.06%

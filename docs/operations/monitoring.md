# Monitoring

Monitoring covers FastAPI health/readiness endpoints, Prometheus-style metrics, gateway snapshots, job queue metrics, and operational runbooks.

## Useful endpoints

- `/api/v2/health`
- `/api/v2/ready`
- `/api/v2/metrics`
- `/api/v2/monitor/summary`

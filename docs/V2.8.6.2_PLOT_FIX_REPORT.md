# V2.8.6.2 绘图系统修复报告

**修复日期**: 2025-10-26  
**版本**: V2.8.6.2  
**修复问题**: MACD histogram显示、MA颜色冲突、MACD策略子图重复

---

## 📋 问题描述

### 1. MACD Histogram 缺失
**现象**: MACD子图中的histogram柱状图不显示或不清晰  
**原因**: 使用 `bt.indicators.MACD()` 时，histogram可能不够明显

### 2. MA线颜色冲突
**现象**: MA均线使用红色或绿色，与K线颜色混淆  
**原因**: 未明确设置MA线的颜色方案

### 3. MACD策略子图重复
**现象**: 运行MACD策略时出现5个子图，其中两个MACD重复  
**原因**: 
- engine.py统一添加MACD绘图指标（1个子图）
- MACD策略内部也创建MACD指标，默认显示（1个重复子图）

---

## 🔧 修复方案

### 1. MACD Histogram 增强显示

**文件**: `src/backtest/engine.py`

```python
# 修改前：使用MACD
self._plot_macd = bt.indicators.MACD(data, subplot=True)

# 修改后：使用MACDHisto确保histogram清晰
self._plot_macd = bt.indicators.MACDHisto(
    data, 
    period_me1=12, 
    period_me2=26, 
    period_signal=9,
    subplot=True
)
```

**说明**: `MACDHisto` 会明确绘制MACD线、信号线和histogram柱状图

### 2. MA线颜色优化

**文件**: `src/backtest/plotting.py`

```python
class CNPlotScheme(PlotScheme):
    def __init__(self):
        super().__init__()
        # K线颜色
        self.barup = 'red'
        self.bardown = 'green'
        # ... 其他配置 ...
        
        # 新增：MA线颜色（避免红绿色）
        self.lcolors = ['blue', 'purple', 'darkorange', 'cyan', 
                       'darkblue', 'magenta', 'gold', 'teal']
```

**说明**: 
- `lcolors` 定义线条颜色循环列表
- Backtrader会按顺序为MA5/10/20/30分配颜色
- 避免使用红色(red)、绿色(green)及相近色系

### 3. MACD策略指标绘图禁用

**文件**: `src/strategies/macd_backtrader_strategy.py`

修改所有MACD策略类的 `__init__` 方法：

```python
# 修改前：
self.macd = bt.indicators.MACD(
    self.data.close,
    period_me1=self.params.fast,
    period_me2=self.params.slow,
    period_signal=self.params.signal,
)

# 修改后：添加 plot=False
self.macd = bt.indicators.MACD(
    self.data.close,
    period_me1=self.params.fast,
    period_me2=self.params.slow,
    period_signal=self.params.signal,
    plot=False  # 关键：避免与engine.py的绘图指标重复
)
```

**影响的策略类**:
- ✅ `MACDStrategy`
- ✅ `MACDZeroCrossStrategy`
- ✅ `MACDHistogramStrategy`
- ✅ `MACD_RegimePullback`
- ✅ `MACD_EnhancedStrategy`

## 🎯 关键技术点

### 1. Backtrader绘图机制

- 指标必须在策略 `__init__` 中创建才能有历史数据
- `subplot=True` 创建独立子图
- `subplot=False, plotmaster=data` 叠加到主图
- `plot=False` 完全不显示

### 2. PlotScheme配置

- `lcolors`: 线条颜色列表（循环使用）
- `barup/bardown`: K线颜色
- `volup/voldown`: 成交量颜色

### 3. 策略包装模式

```python
# 动态创建子类
class StrategyWithPlotIndicators(original_strategy_cls):
    pass

# 临时替换
module.strategy_cls = StrategyWithPlotIndicators
module.add_strategy(cerebro, params)
module.strategy_cls = original_strategy_cls  # 恢复
```

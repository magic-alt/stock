## ✨ 主要变化

### 1. 架构重构
```
V1.0 (单文件)          →    V2.0 (模块化)
├── stock_monitor.py   →    ├── main.py (主程序)
├── quick_monitor.py   →    └── src/ (模块包)
└── quick_backtest.py  →        ├── data_sources/ (数据)
                                ├── indicators/ (指标)
                                ├── strategies/ (策略)
                                ├── backtest/ (回测)
                                ├── monitors/ (监控)
                                └── utils/ (工具)
```

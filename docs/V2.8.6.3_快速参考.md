## 📊 自动生成的分析结果

### 1. Pareto 前沿 CSV
**路径**: `cache/grid_analysis/{strategy}_pareto.csv`

**内容**: 
- 非劣解（Pareto 最优配置）
- 基于 Sharpe、收益率、最大回撤筛选
- 包含完整的交易统计数据

**使用场景**:
- 快速找到最优参数组合
- 理解收益率与风险的权衡
- 导入 Excel 进行进一步分析

### 2. 参数热力图 PNG
**路径**: `cache/grid_analysis/{strategy}/heat_{strategy}.png`

**内容**:
- 参数组合 vs 收益率的 2D 可视化
- 灰色区域 = 无交易或失败的组合
- 颜色深浅 = 收益率高低

**支持策略**:
| 策略 | 热力图类型 | 参数维度 |
|------|-----------|---------|
| MACD | fast vs slow | 2D |
| Bollinger | period vs devfactor | 2D |
| RSI | period vs upper | 2D |
| EMA | period vs return | 1D 曲线 |
| ZScore | z_entry vs z_exit | 2D |
| Triple MA | fast vs mid | 2D |

### 自动分析集成点

```python

# → 自动调用 pareto_front() + save_heatmap()
```

## 🚀 使用示例

### 示例 1: 单策略网格搜索（自动分析）

```python
from backtest.engine import BacktestEngine

engine = BacktestEngine(cache_dir="./cache")

# 执行网格搜索
result_df = engine.grid_search(
    strategy="macd",
    grid={"fast": [8, 10, 12], "slow": [20, 24, 26], "signal": [7, 9]},
    symbols=["000858.SZ", "600036.SH"],
    start="2023-01-01",
    end="2023-12-31",
    cash=200000,
    commission=0.0003,
    max_workers=4,  # 并行加速
)

# 控制台输出：
# ✅ Pareto frontier saved: ./cache/grid_analysis/macd_pareto.csv (5 points)
# [macd] zero-trade cells: 2.5%
# ✅ Heatmaps saved in: ./cache/grid_analysis/macd

# 查看最优配置
top_5 = result_df.nlargest(5, "sharpe")
print(top_5[["fast", "slow", "signal", "sharpe", "cum_return", "mdd"]])
```

## ❓ 常见问题

### Q1: 为什么 Pareto 前沿只有 1 个点？
A: 当前参数组合中只有 1 个非劣解。其他配置在 Sharpe、收益率、回撤至少一个指标上被该配置支配。

**解决方法**: 扩大网格搜索范围，增加参数多样性。

### Q2: 热力图全是灰色怎么办？
A: 灰色区域表示无交易或交易失败（trades <= 0）。

**原因**:
- 参数范围不合理（例如 fast >= slow 的 MACD）
- 数据周期太短，策略无法产生信号
- 策略逻辑本身有问题

**解决方法**: 
1. 检查参数范围的合理性
2. 延长回测周期
3. 调试策略逻辑

### Q3: 如何禁用自动分析？
A: 当前版本不支持禁用。分析失败时会自动跳过并输出警告，不影响 grid_search 主流程：

```
⚠️  Failed to generate analysis outputs: {error_message}
```

### Q4: 分析文件占用空间大吗？
A: 占用空间很小：
- Pareto CSV: 通常 < 10 KB
- 热力图 PNG: 通常 10-30 KB

整个 `cache/grid_analysis/` 目录通常 < 1 MB。

## 🚀 单只股票回测（新功能）

最快速的回测流程，适合单只股票快速分析：

```bash
# 1. 启动 GUI
python backtest_gui.py

# 2. 选择单只股票（一键操作）
数据配置 → 快速选择 → 单股: [茅台]
# 自动填充: 600519.SH

# 3. 下载数据（首次使用）
[📥 下载数据]
# 等待30秒-1分钟

# 4. 选择策略
策略配置 → 双击选择: ema, macd

# 5. 开始回测
控制区域 → [▶️ 开始回测]

# 6. 查看结果
reports_gui/
  ├── ema_chart.png      ← 📊 图表
  ├── ema_nav.csv        ← 📈 净值数据
  ├── macd_chart.png
  └── macd_nav.csv
```

**预计耗时**: 2-3分钟（含下载）

## ⚠️ 注意事项

### 数据下载

- 首次下载需要网络连接
- 数据量较大时可能需要几分钟
- 下载失败可重试或检查网络

### 图表生成

- 图表默认开启，可在"回测配置"中关闭
- 图表保存为 PNG 格式，高分辨率
- 多策略回测会生成多张图表

### 单只股票回测

- 单只股票回测速度快（通常1-2分钟）
- 适合快速测试策略效果
- 建议先单股测试，满意后再批量

## 📞 常见问题

### Q1: 为什么下载数据失败？

A: 检查网络连接，确保能访问数据源（akshare/tushare）

### Q2: 单只股票按钮找不到我要的股票？

A: 直接在输入框中输入股票代码，每行一个

### Q3: 图表在哪里？

A: `reports_gui/` 目录，文件名为 `{策略名}_chart.png`

### Q4: 能同时测试多只股票吗？

A: 可以，在输入框中每行输入一个股票代码，或使用组合按钮

### Q5: 下载的数据存在哪里？

A: `./cache/` 目录（可在"数据配置"中修改）

## 🚀 使用流程

### 最简流程（3步）
```
1. 双击 启动GUI.bat
2. 点击 "白酒股" + "趋势策略" 
3. 点击 "开始回测"
```

### 标准流程（5步）
```
1. 配置数据（股票代码 + 日期）
2. 选择策略（多选或快捷按钮）
3. 设置回测参数（资金、费率）
4. 选择运行模式（推荐：自动化流程）
5. 点击开始回测
```

### 完整流程（7步）
```
1. 配置数据源和股票代码
2. 点击"预览数据"验证
3. 选择策略并配置参数
4. 查看策略详情
5. 设置回测和优化参数
6. 保存配置（可选）
7. 开始回测并查看日志
```

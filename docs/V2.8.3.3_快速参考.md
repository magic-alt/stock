## MACD 策略对比

| 特性 | macd (原版) | macd_e (增强版) |
|------|------------|----------------|
| 进出场信号 | MACD 金叉/死叉 | MACD 金叉/死叉 + 趋势过滤 |
| 趋势过滤 | ❌ 无 | ✅ EMA200 向上才做多 |
| 冷却期 | ❌ 无 | ✅ 平仓后 5 bar 休息 |
| 最小持仓 | ❌ 无 | ✅ 至少持仓 3 bar |
| 止损止盈 | ❌ 无 | ✅ 5% 止损 / 10% 止盈 |
| 适用场景 | 趋势市 | 震荡市 + 趋势市 |
| 交易频率 | 高 | 低（减少 93%）|

## 使用示例

```bash
# 原版 MACD（图表优化，逻辑不变）
python unified_backtest_framework.py run --strategy macd --symbols 600519.SH --start 2024-01-01 --end 2024-12-31 --plot

# 增强版 MACD（默认参数）
python unified_backtest_framework.py run --strategy macd_e --symbols 600519.SH --start 2024-01-01 --end 2024-12-31 --plot

# 增强版 - 关闭趋势过滤
python unified_backtest_framework.py run --strategy macd_e --symbols 600519.SH --start 2024-01-01 --end 2024-12-31 --params "trend_filter=False" --plot

# 增强版 - 调整止损止盈
python unified_backtest_framework.py run --strategy macd_e --symbols 600519.SH --start 2024-01-01 --end 2024-12-31 --params "stop_loss_pct=0.03,take_profit_pct=0.15" --plot

# 增强版 - 网格搜索
python unified_backtest_framework.py grid --strategy macd_e --symbols 600519.SH --start 2023-01-01 --end 2024-12-31
```

## 增强策略参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| `fast` | 12 | 8-20 | MACD 快线周期 |
| `slow` | 26 | 20-40 | MACD 慢线周期 |
| `signal` | 9 | 固定 | MACD 信号线周期 |
| `ema_trend_period` | 200 | 100-200 | 趋势过滤均线 |
| `trend_filter` | True | True/False | 是否启用趋势过滤 |
| `cooldown` | 5 | 3-8 | 平仓后冷却期（bar）|
| `min_hold` | 3 | 2-5 | 最小持仓期（bar）|
| `stop_loss_pct` | 0.05 | 0.03-0.08 | 止损百分比 |
| `take_profit_pct` | 0.10 | 0.07-0.15 | 止盈百分比 |

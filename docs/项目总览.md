## 📈 技术指标说明

### MA (移动平均线)
- **MA5**: 5日均线 - 短期趋势
- **MA10**: 10日均线 - 中期趋势
- **MA20**: 20日均线 - 长期趋势

### RSI (相对强弱指标)
- **> 70**: 超买区域，可能回调
- **< 30**: 超卖区域，可能反弹
- **30-70**: 正常区域

### MACD (指数平滑异同移动平均线)
- **金叉**: MACD上穿Signal线，看涨
- **死叉**: MACD下穿Signal线，看跌

## 🔧 回测策略

### 策略1: 双均线交叉
- **买入**: MA5上穿MA20
- **卖出**: MA5下穿MA20
- **特点**: 趋势跟随，适合趋势明显的市场

### 策略2: RSI超买超卖
- **买入**: RSI < 30（超卖）
- **卖出**: RSI > 70（超买）
- **特点**: 逆向交易，适合震荡市场

## 💡 使用技巧

### 1. 首次使用
先运行 `test_connection.py` 确保数据连接正常

### 2. 监控多只股票
在config.py中添加更多股票代码，但建议不超过10只，以免刷新太慢

### 3. 交易时间外
非交易时间显示的是上一个交易日的数据

### 4. 数据延迟
免费数据源通常有15分钟延迟，这是正常的

### 5. 回测参数
- 短期回测：30-90天
- 中期回测：180-365天
- 长期回测：1-3年

## ⚠️ 注意事项

1. **数据延迟**: A股免费数据通常有15分钟延迟
2. **刷新频率**: 建议不低于30秒，避免频繁请求
3. **交易时间**: 周一至周五 9:30-15:00（午休11:30-13:00）
4. **网络要求**: 需要稳定的网络连接
5. **仅供参考**: 所有数据和分析不构成投资建议

## 🐛 故障排除

### 问题1: 无法获取数据
- 检查网络连接
- 运行 `test_connection.py` 测试
- 确认股票代码正确

### 问题2: 显示乱码
- Windows用户确保使用 UTF-8 编码
- 或通过 `启动工具.bat` 运行

### 问题3: 程序运行缓慢
- 减少监控的股票数量
- 增加刷新间隔
- 使用 `quick_monitor.py` 代替完整版

### 问题4: 导入错误
确保已安装依赖：
```bash
pip install akshare pandas numpy
```

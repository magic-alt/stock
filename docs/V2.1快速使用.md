## 🆘 常见问题

### Q1: 为什么股数是300而不是354？
**A:** A股规定最小交易单位为100股（1手），系统自动向下取整到100的整数倍。

### Q2: 成交额为什么有时显示"万"有时显示"亿"？
**A:** 自动转换，≥1亿显示"亿"，<1亿显示"万"。

### Q3: Backtrader图表不显示怎么办？
**A:** 确保安装了依赖：
```bash
pip install backtrader matplotlib
```

### Q4: 资金不够买1手怎么办？
**A:** 系统会跳过该交易，不会强制买入。

### Q5: 简单引擎和Backtrader引擎结果不同？
**A:** 正常现象，两者实现方式略有差异，但都遵循A股交易规则。

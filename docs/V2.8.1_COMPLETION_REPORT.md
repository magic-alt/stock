## 🔧 修复内容总结

### 1. 基准指数加载错误 ✅

**问题**: 
- `KeyError: 'date'` - 缓存文件格式不兼容
- akshare API 期望 `sh000300` 格式，而非 `000300.SH`

**修复**:
```python
# src/data_sources/providers.py 第268-325行

# 1. 添加指数代码格式转换
if '.' in index_code:
    symbol, exchange = index_code.split('.')
    if exchange.upper() in ['SH', 'SS']:
        ak_index_code = f'sh{symbol}'  # 000300.SH -> sh000300
    elif exchange.upper() == 'SZ':
        ak_index_code = f'sz{symbol}'  # 399001.SZ -> sz399001

# 2. 改进缓存读取（向后兼容）
if os.path.exists(cache_file):
    try:
        df = pd.read_csv(cache_file, index_col=0, parse_dates=[0])  # 位置索引
        if df.index.name != 'date':
            df.index.name = 'date'
    except Exception:
        df = pd.read_csv(cache_file)
        df = _standardize_index_frame(df)  # 回退方案
```

**验证结果**:
```
✅ 成功加载 22 条数据
✅ 索引名称: date
✅ 无 KeyError
```

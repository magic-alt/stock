## 已知问题与解决

### 1. 基准指数加载警告 ⚠️

**现象**:
```
UserWarning: Failed to load index 000300.SH: 'date'. Using flat NAV.
```

**影响**: 低（使用平坦基准代替）
**原因**: 基准数据格式问题或日期范围不匹配
**解决**: 
- 可忽略（不影响回测）
- 或使用 `--benchmark 000300.SS` (不同数据源)
- 或不指定基准

### 2. PowerShell JSON 转义问题 ✅ 已解决

**原问题**: 
```bash
--grid '{"period":[10,20,30]}'  # 失败
```

**解决方案**:
```bash
# 方案 1: 使用默认网格（推荐）
python unified_backtest_framework.py grid --strategy ema --symbols 600519.SH --start 2024-01-01 --end 2024-12-31

# 方案 2: 创建配置文件
# 1. 创建 grid_config.json
# 2. 策略模块中读取（需要代码修改）
```

### 3. 短期数据无交易信号 ℹ️

**现象**: 3个月数据可能无交易
**原因**: 策略参数期待更长周期
**解决**: 使用至少 6-12 个月数据进行测试

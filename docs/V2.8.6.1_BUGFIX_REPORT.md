## 🐛 Bug #1: 佣金显示错误 (100倍)

### 问题描述

**用户报告**：
```
2023-11-07, BUY EXECUTED, Size 100, Price: 1805.90, Cost: 180590.25, Commission 0.1806
                                                                                    ^^^^
                                                                                    错误！
```

**正确应为**：
- 100股 @ 1805.90元 = 180590元
- 佣金 (0.01%) = 180590 × 0.0001 = **18.06元** ✅
- 实际显示 = **0.18元** ❌ (相差100倍)

### 根本原因

**Backtrader 的 `order.executed.comm` 存储方式存在不一致性**：

| 订单类型 | 存储内容 | 实际值 (100股@159元) |
|---------|---------|---------------------|
| **买入** | 未知格式 (不可靠) | 0.0477 (无法直接使用) |
| **卖出** | 混合格式 (不可靠) | 23.897 (印花税绝对值+佣金百分比) |

**初步尝试**：
- 第一次修复尝试：将格式从 `.2f` 改为 `.4f` → **无效** ❌
- 第二次调查：尝试 `order.executed.comm * abs(order.executed.size)` → **错误放大** ❌

**最终发现**：
```python
# 买入测试 (100股 @ 159元)
order.executed.comm = 0.0477  # 不是每股佣金！
order.executed.comm * 100 = 4.77  # 正确！

# 卖出测试 (300股 @ 157.42元)
order.executed.comm = 23.897  # 不是每股费用！
order.executed.comm * 300 = 7169.1  # 错误！(应该是28.34)
```

**结论**: `order.executed.comm` **不可靠**，无法直接使用！

### 解决方案

**放弃使用 `order.executed.comm`，直接从 `order.executed.value` 计算**：

```python
# 获取交易金额
value = abs(order.executed.value)

# 根据交易类型计算费用
if order.isbuy():
    # 买入：仅佣金 (0.01%)
    total_cost = value * 0.0001
else:
    # 卖出：佣金 (0.01%) + 印花税 (0.05%) = 0.06%
    total_cost = value * 0.0006

# 显示
print(f"Commission {total_cost:.2f}")
```

## 🐛 Bug #2: 空白图表问题

### 问题描述

**用户报告**：
> 当terminal显示"正在生成图表..."时会弹出两张figure，其中figure1为空白图表

**症状**：
- 运行 `--plot` 参数时，会弹出两个matplotlib窗口
- 第一个窗口 (Figure 1) 是空白的
- 第二个窗口包含正常的K线图和指标

### 根本原因

**Backtrader的 `cerebro.plot()` 实现问题**：
- 在绘图过程中可能创建临时/占位的 matplotlib figure
- 这些 figure 没有 axes (坐标轴)，也没有实际内容
- 但它们仍然被显示出来，造成用户困扰

### 解决方案

**在 `cerebro.plot()` 前后检测并关闭空白图表**：

```python
# 1. 记录绘图前的figure数量
figs_before = set(plt.get_fignums())

# 2. 执行绘图
figs = cerebro.plot(**plot_kwargs)

# 3. 检测新创建的figure
figs_after = set(plt.get_fignums())
new_figs = figs_after - figs_before

# 4. 识别有效图表 (包含axes的figure)
if len(new_figs) > 1:
    valid_figs = []
    for fignum in new_figs:
        fig = plt.figure(fignum)
        if fig.get_axes():  # 检查是否有坐标轴
            valid_figs.append(fignum)
    
    # 5. 关闭空白图表
    for fignum in new_figs:
        if fignum not in valid_figs:
            plt.close(fignum)
    
    if len(new_figs) - len(valid_figs) > 0:
        print(f"[OK] 已自动关闭 {len(new_figs) - len(valid_figs)} 个空白图表")
```
